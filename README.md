## 系统文件操作库
一个通过本地浏览器即可操作系统本地文件的JavaScript库,支持文件上传、下载、删除、重命名、复制、移动、预览等操作。
对文本数据的批量处理操作,代替传统的文本处理软件,只需一个浏览器,全平台通用,无需下载.

## 安装

1. **克隆或下载本仓库**：

   ```bash
   git clone 
   ```

2. **进入项目目录**：

   ```bash
   cd 
   ```

3. **安装依赖包**：

   ```bash
   npm install
   ```

4. **构建**：

   ```bash
   npm run build
   ```

5. **运行**：

   ```bash
   npm start
   ```
